console.log

Пример использования команды console.log:
console.log('Текст, который будет выведен в консоль');

Alert()

Пример использования команды alert:
alert('Текст, который будет выведен в модальное окно');

Prompt()

Пример использования команды prompt:
prompt('Текст, который будет выведен в модальное окно', [Необязательный текст в поле ввода по умолчанию]);
